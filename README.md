
# Api para cadastro de alunos 🎓 🚀- Desafio 2 Rocketseat 

Foi utilizado:

* Express,
* NodeJS,
* Docker,
* Sucrase,
* Nodemon,
* ESLint,
* Prettier,
* EditorConfig,
* Sequelize com o PostgreSQL,
* JWT,
* Yup;,

É possível inserir informações dos estudantes e atualizá las, mas apenas o usuário admin pode ter esse privilégio.
